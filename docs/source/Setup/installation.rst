============================
Installation
============================

To use unisphere, you will need to first install it from the Github repository:

.. code-block:: console

  git clone https://github.com/UniSpheree/unisphere-app.git
  cd unisphere-app

Then you must initialize the backend:

.. code-block:: console

   cd backend
   npm install
   npm start

Finally, install the required packages and launch the app:

.. code-block:: console

   flutter pub get
   flutter run -d chrome
